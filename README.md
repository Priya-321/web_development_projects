# Project 1: Color Palette

# Description
This is an interactive Color Palette Generator that allows users to explore and copy color codes from pre-defined color palettes. The project features 12 beautifully curated color palettes, each containing five shades representing a specific theme such as Sunset, Nature, Forest, Ocean, Candy, and more. Designed with flexibility and responsiveness in mind, this project is an excellent tool for designers, developers, or anyone looking to experiment with colors.

Key Features:
  Pre-defined Color Palettes: A total of 12 palettes, each with five carefully selected shades.
  Responsive Design: The layout adapts seamlessly to different screen sizes, making it accessible on mobile, tablet, and desktop devices.
  Copy Color Code: Users can click on a shade to copy its HEX color code to the clipboard, making it easy to use in other projects.
  Hover Effects: Interactive buttons and sections with smooth transitions for an engaging user experience.
  Customizable Palettes: Palettes can be easily updated or extended by modifying the JavaScript object storing the color data.


Technologies Used:    
  HTML: For the basic structure of the project.
  CSS: For styling the interface, with responsive designs using media queries.
  JavaScript: To handle dynamic palette generation, interactive color copying, and event handling.

Responsive Design:
Mobile: Stacks sections vertically for a compact view.
Tablet: Adapts to display two sections per row for improved readability.
Desktop: Utilizes horizontal space to showcase all sections in a row.

Customization:
To add or update palettes:
Open the script.js file.
Modify the colorPalettes object by adding new palettes or updating existing ones.

Future Improvements:
Add a "Create Your Own Palette" feature.
Integrate support for different color formats like RGB and HSL.
Include a color contrast checker for accessibility.


This project is perfect for practicing responsive web design, event handling in JavaScript, and understanding color theory. Contributions and feedback are welcome!
