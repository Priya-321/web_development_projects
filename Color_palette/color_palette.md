#Color Palette Project
A responsive and interactive color palette generator built with HTML, CSS, and JavaScript. This project allows users to explore multiple predefined color palettes and their family shades, with an intuitive UI that adapts seamlessly to different devices. Each color section includes a button for users to copy the color code directly to the clipboard.

#Features
Predefined Palettes:
Includes 12 color palettes such as Sunset, Nature, Ocean, Candy, Forest, and more.
Each palette contains 5 shades within its color family.

#Interactive UI:
Click on a palette button to load its colors dynamically.
Each color section features a copy button to save the color code directly to the clipboard.

#Responsive Design:
The layout adapts to mobile, tablet, and desktop screens using CSS media queries.
Flexible arrangement of color sections for better usability on all devices.
Smooth Animations:
Hover effects on palette buttons for an engaging user experience.

#Customization:
Easily add new palettes and modify existing ones by editing the colorPalettes object in the JavaScript file.

#Technologies Used
HTML: For structure.
CSS: For styling, animations, and responsive design.
JavaScript: For dynamic updates, event handling, and clipboard functionality.

#How to Use
Clone or download the repository.
Open the index.html file in your browser.
Click on any palette button to view the shades.
Click the copy button inside each color section to save its hex code.

#Project Structure
index.html: The main HTML file containing the UI structure.
styles.css: The CSS file for styling and responsiveness.
script.js: The JavaScript file for interactive functionality.
